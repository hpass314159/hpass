Basic usage

  1. Enter password seed in blue box
  2. Click "Generate and copy" button
  3. Paste generated password from clipboard

Icons

  • help - shows this page
  • hide/show - toggle visibility
  • info - in depth background

Options

You should change default parameters (in parentheses below) in order to generate passwords unique to you. The values you enter are saved locally and reused between sessions. NOTE: These saved values will be erased when browser cache/cookies are cleared.
  1. Pepper (!) - special character is recommended as these are required by most sites
  2. Salt (SALT) - any secret you want
  3. Length (15) - longer passwords are more secure, but some sites are limiting length. Default value 15 seems to be accepted by most sites.
  4. Burnin (9) - the number of initial (throw away) random number generator cycles
