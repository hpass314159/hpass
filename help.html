Press button to save these values on your device. Password prompt entered in "enter hint" box at the top of the screen together with these settings determines the generated password i.e. with the same values always the same password will be generated. NOTE: it is important to store these values in some safe place in case e.g. you loose your device etc. In this case, if you install the app on the new device you would need to enter these settings in order to create the same password as on the old device.

Basic usage

  1. Enter password hint in enter hint box. It is recommended to use as a hint simply secondary domain of the website which asks for password e.g. netflix. Note: you can simply copy and paste https://www.netflix.com/login and with Clean option set to true it will extract just netflix as a hint.
  2. Click button to generate password and password will be automaticall copied to the clipboard.
  3. Paste generated password from clipboard where it is needed.
