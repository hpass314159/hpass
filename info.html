What is it?

Simple to use generator to create strong passwords from easy to remember hints.

Rationale

One way to securely store and share passwords is by using a password manager. It is an often recommended and widely used solution, but not without problems.

HPASS(™) uses an alternative approach with two main advantages:

  1. Passwords are generated locally, on demand and never stored - there is no danger of data breach.
  2. There is no master password to remember.

Large number of sites and apps offer strong random password generation, but despite some (not exhaustive!) search (see some links below) I was not able to locate any which would allow me to generate reproducible, hint-based passwords.

Simple Example

The table below illustrates how from the same hint, using easily customizable options passwords of varied complexity can be generated. As an example, let’s assume Alice and Bob want to create individual passwords to access their respective netflix accounts. As a hint they may use, well… netflix, and as a secret they may use their first names (with default prefix=’:’, burnin=8, length=15) 

hint secret generated password
netflix alice WN1KUubX94:rhs8
netflix bob Z1E:QAcP8wYjpMX


Of course, they can use anything else as a secret and change other parameters as well as they wish. Please note that the way HPASS generates password from given input parameters (hint, sekret, prefix, burnin and length) is one-way hash function and the input parameters cannot be recovered given the password string (see some more elaboration see: https://www.unblocktalent.com/topics/building-blocks/is-hashing-reversible/). Someone who would want to generate your password for any specific account would have to know all these parameters AND the source code of the hashing function.

Problems with password managers

You can find multiple sources discussing pros and cons of using password manager. Very short list is:

  • Pros: convenience, ability to share passwords
  • Cons: need to remember master password, vulnerability to hacking, safety

We can argue that HPASS has all adantages and none of dissadventages of password managers.
